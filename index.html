Resultado:

Para continuar o exercício, siga as instruções abaixo:

  1. Subtração:
    • Insira dois números nos campos "Input 1" e "Input 2".
    • Clique no botão "Subtrair".
    • O resultado da subtração será exibido no parágrafo com o id "resultado".
  2. Multiplicação:
    • Insira dois números nos campos "Input 1" e "Input 2".
    • Clique no botão "Multiplicar".
    • O resultado da multiplicação será exibido no parágrafo com o id "resultado".
  3. Divisão:
    • Insira dois números nos campos "Input 1" e "Input 2".
    • Clique no botão "Dividir".
    • Se o valor do "Input 2" for 0, uma mensagem de erro será exibida no parágrafo com o id "resultado".
    • Se o valor do "Input 2" não for 0, o resultado da divisão será exibido no parágrafo com o id "resultado".
